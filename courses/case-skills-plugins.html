andrej-karpathy-skills

github.com/multica-ai/andrej-karpathy-skills · ⭐ 165k

github.com/kerlos/pordee

Global vs Project Scope — เรื่องที่คนมักสับสน

ก่อนลง plugin ต้องตัดสินใจว่าอยากใช้แค่โปรเจกต์เดียวหรืออยากใช้ทุกที่ เพราะตำแหน่งที่วางไฟล์จะต่างกัน

# โครงสร้างโฟลเดอร์ที่ plugin ไปวางตัว
C:\Users\Admin\
├── .claude\                    ← USER SCOPE (ทุก project)
│   ├── CLAUDE.md
│   └── plugins\
│       ├── andrej-karpathy-skills\
│       └── pordee\
└── myproject\                  ← PROJECT SCOPE
    └── .claude\
        └── CLAUDE.md           ← กฎเฉพาะโปรเจกต์นี้
Scopeชื่อในหน้า installใช้ได้กับเหมาะกับ
User Install for you (user scope) ทุก project Plugin ส่วนตัว
Project Install for all collaborators โปรเจกต์นี้ + ทีม กฎของทีม
Local Install for you, in this repo only โปรเจกต์นี้คนเดียว ทดลองก่อนแชร์
🌍
USER SCOPE
เปิด CMD ที่ไหนก็ได้ plugin ติดมาเสมอ
📁
PROJECT SCOPE
ต้องเปิด CMD ในโฟลเดอร์โปรเจกต์
🔒
LOCAL SCOPE
เหมือน project แต่ไม่ commit ขึ้น git
🤔
คำถามช่วยตัดสินใจ
อยากใช้ทุก project? → เลือก User scope
อยากใช้แค่โปรเจกต์เดียว? → เลือก Project หรือ Local scope

วิธีติดตั้ง Step by Step

Plugin ติดตั้งผ่าน Claude Code CLI โดยใช้คำสั่ง /plugin install — ต้องมี Node.js และ Claude Code พร้อมก่อนถึงจะลงได้

1. ติดตั้ง Node.js

# ดาวน์โหลดจาก nodejs.org แล้ว install ตามปกติ
# ตรวจว่าได้แล้ว:
node --version

2. ติดตั้ง Claude Code CLI

npm install -g @anthropic-ai/claude-code

3. เปิด CMD แล้วรัน Claude Code

claude

กด Enter เพื่อ trust folder เมื่อถูกถาม รอจนพร้อมใช้งาน

4. เพิ่ม andrej-karpathy-skills เข้า marketplace แล้วลง

/plugin marketplace add forrestchang/andrej-karpathy-skills
/plugin install andrej-karpathy-skills@karpathy-skills

5. เพิ่ม pordee เข้า marketplace แล้วลง

/plugin marketplace add kerlos/pordee
/plugin install pordee@pordee

6. เลือก Scope เมื่อถูกถาม

💡
เลือก "Install for you (user scope)" เพื่อใช้ได้ทุกโปรเจกต์โดยไม่ต้องลงซ้ำ

7. ตรวจสอบว่าลงครบ

/plugin list
✅
เห็น andrej-karpathy-skills และ pordee ใน list = ลงสำเร็จ
5

หลังลงแล้ว — ไฟล์อยู่ที่ไหน

Plugin ที่ลง user scope จะไปวางในโฟลเดอร์ .claude ของ user ซึ่งเป็น โฟลเดอร์ซ่อน ต้องเปิด show hidden files ถึงจะเห็น

C:\Users\Admin\.claude\         ← โฟลเดอร์ซ่อน (File Explorer: View → Hidden items)
├── plugins\
│   ├── andrej-karpathy-skills\
│   └── pordee\
└── skills\                     ← skills ที่สร้างเองวางที่นี่

เปิดดูโฟลเดอร์ได้ทันทีด้วย:

explorer C:\Users\Admin\.claude
6

Tips & Commands

Commands ที่ใช้บ่อยสำหรับจัดการ plugin:

# ดู plugin ที่ลงไว้ทั้งหมด
/plugin list
# ลบ plugin ที่ไม่ต้องการ
/plugin remove [ชื่อ-plugin]
# อัปเดต Claude Code เป็น version ล่าสุด
npm update -g @anthropic-ai/claude-code
# ล้างเฉพาะ plugins (เก็บ skills/settings ไว้)
rmdir /s /q C:\Users\Admin\.claude\plugins
# ล้างทุกอย่างใน .claude (reset หมด)
rmdir /s /q C:\Users\Admin\.claude
⚠️
ระวัง: rmdir /s /q .claude ลบ ทุกอย่าง รวม CLAUDE.md ส่วนตัว skills และ settings — ใช้เฉพาะตอน reset จริงๆ

🎯 สิ่งที่ได้จากเคสนี้

  • Plugin ติดตั้งครั้งเดียว — Claude ทำตามกฎอัตโนมัติทุก session ไม่ต้องพิมซ้ำ
  • เลือก User scope เมื่ออยากใช้ทุกโปรเจกต์, Project scope เมื่ออยากแชร์ทีม
  • karpathy-skills: โค้ดมีวินัย — Think Before Coding / Simplicity / Surgical / Goal-Driven
  • pordee: ตอบไทยกระชับ ประหยัด token 60–80%, เปิด/ปิดได้ทันทีด้วย keyword
  • ไฟล์อยู่ที่ C:\Users\Admin\.claude\plugins\ (โฟลเดอร์ซ่อน)
