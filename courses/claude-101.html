Claude 101

รู้จัก Claude และใช้มันเป็น thinking partner ในงานประจำวัน — ตั้งแต่การคุยให้ได้ผลดี ไปจนถึงจัดระเบียบงานด้วย Projects/Artifacts/Skills และต่อ Claude เข้ากับเครื่องมือที่ใช้จริง

⏱️ 14 บท · ระดับเริ่มต้น 🎯 ใช้ได้ทั้ง web / desktop / mobile ✅ สถานะ: เรียนจบแล้ว

คอร์สนี้ไล่จาก "Claude คืออะไรและคุยอย่างไรให้ได้ผล" → จัดระเบียบงานและความรู้ → ต่อ Claude เข้ากับเครื่องมือและค้นข้อมูลเชิงลึก → จนถึงการเอาไปใช้จริงตามบทบาทงาน เรียบเรียงตามลำดับบทเรียนจริงด้านล่าง

👋 รู้จัก Claude และการคุยครั้งแรก

Claude คือ AI assistant ที่ออกแบบให้เป็น thinking partner ยึดหลัก helpful / harmless / honest ผ่านแนวทาง Constitutional AI ทำได้หลายงาน ทั้งสรุป ค้นหา เขียน ตอบคำถาม และเขียนโค้ด เข้าถึงได้ทั้งบนเว็บ เดสก์ท็อป และมือถือ

เคล็ดลับการคุยครั้งแรก: คุยกับมันเหมือน เพื่อนร่วมงาน โดยจัดบริบทให้ครบ — setting the stage (ภูมิหลัง), define the task (งานที่อยากได้), และ specify rules (ข้อกำหนด/รูปแบบ) การอัปโหลดเอกสารช่วยให้ Claude เข้าใจงานได้ดีขึ้น และพลังจริง ๆ มาจาก การคุยต่อเนื่อง ปรับไปทีละรอบ ไม่ใช่สั่งครั้งเดียวจบ

Getting better results

บทนี้รวมวิธีแก้ปัญหาที่มือใหม่เจอบ่อย แนะนำให้ศึกษาแนวคิด AI Fluency ควบคู่ และตั้ง evals (ชุดทดสอบเล็ก ๆ) เพื่อดูว่า Claude ทำงานเข้ากับ workflow ของเราได้ดีแค่ไหน

▶ วิดีโอ: แก้ปัญหาที่พบบ่อย + แนะนำ evals

โหมด Chat, Cowork, Code บนเดสก์ท็อป

แอป Claude บนเดสก์ท็อปมี 3 โหมดให้เลือกตามงาน: Chat (ถาม/ร่างงานเร็ว ๆ), Cowork (งานหลายขั้นตอนบนไฟล์จริง), และ Code (งานพัฒนา) พร้อมลูกเล่นอย่าง quick entry และ scheduled tasks

🗂️ จัดระเบียบงานและความรู้

📁

Projects

workspace ที่มี memory/knowledge/instructions อัปโหลดเอกสารไว้ให้อ้างอิงได้ทุกแชท เมื่อ knowledge ใหญ่ขึ้นจะเปิด RAG อัตโนมัติ (อ้างอิงได้กว้างขึ้นมาก) และแชร์ให้ทีมได้

🎨

Artifacts

ผลลัพธ์ที่เป็นชิ้นงาน standalone และโต้ตอบได้ (เว็บ/chart/เอกสาร) เปิดในหน้าต่างแยก สร้างอัตโนมัติเมื่อเนื้อหายาวพอ และแชร์/เผยแพร่ได้

📦

Skills

โฟลเดอร์รวม instructions/scripts/resources ที่ Claude โหลดมาใช้แบบ dynamic มี built-in skills สำหรับสร้างเอกสาร เปิดใช้ได้ใน settings

▶ วิดีโอ: Skills ในผลิตภัณฑ์ Claude

🔗 ต่อ Claude เข้ากับเครื่องมือและค้นข้อมูลเชิงลึก

Connectors ให้ Claude เข้าถึงเครื่องมือ/ข้อมูลที่คุณใช้ทุกวัน — ไม่ใช่แค่อ่านข้อมูล แต่ทำ action ได้ด้วย (ค้น ดึง วิเคราะห์ สร้าง อัปเดต) ตามสิทธิ์ที่ให้ ส่วน Enterprise search ค้นข้ามแหล่งข้อมูลขององค์กรโดยมี security/permissions ปกป้องข้อมูล (สำหรับ Team/Enterprise ที่ admin เปิดใช้)

Research mode ทำงานแบบ agentic — ค้นหลายรอบต่อเนื่อง สำรวจหลายมุมโดยอัตโนมัติ แล้วสรุปเป็นรายงาน ใช้เวลาราว 5–15 นาที (งานซับซ้อนถึง ~45 นาที)

🎯 เอาไปใช้จริง

คอร์สปิดท้ายด้วยตัวอย่าง use-case ตามบทบาทงาน (พร้อมไกด์ละเอียดให้ลองทันที) และแนะนำผลิตภัณฑ์เฉพาะทางอื่น ๆ ในตระกูล Claude เช่น Claude Code และ Claude สำหรับ Slack/Excel/PowerPoint/Chrome

🎯 สรุปใจความสำคัญ

  • Claude เป็น thinking partner ที่ยึดหลัก helpful/harmless/honest ผ่าน Constitutional AI
  • คุยให้ได้ผล = ให้บริบทครบ (เวที + งาน + กฎ) และปรับต่อเนื่องทีละรอบ
  • จัดระเบียบงานด้วย Projects (มี RAG อัตโนมัติเมื่อความรู้ใหญ่ขึ้น), Artifacts และ Skills
  • ขยายขอบเขตด้วย Connectors, Enterprise search และ Research mode (agentic)
  • เลือกโหมด Chat / Cowork / Code ให้ตรงกับลักษณะงาน
