Case Study: Claude Cowork
รวมข้อมูลหลายไฟล์ → รายงานสรุปในครั้งเดียว

โจทย์: มีไฟล์ยอดขายหลายเดือนกระจัดกระจาย อยากได้ รายงานสรุปรายไตรมาส พร้อมไฮไลต์ — ให้ Cowork ทำงานหลายขั้นบนไฟล์จริงจนได้ผลเป็น artifact

🎯 ใช้กับ Claude Desktop (Cowork)🧩 อิงคอร์ส Introduction to Claude Cowork📋 step-by-step
🧭
สถานการณ์ งานนี้เหมาะกับ Cowork เพราะเป็นงาน หลายขั้นตอน + อิงไฟล์จริง — Cowork จะวางแผน ลงมือ และส่งมอบเป็นไฟล์ให้รีวิวเหมือนรับงานจากเพื่อนร่วมงาน
1

เตรียมโฟลเดอร์ทำงานเฉพาะ

แนวปฏิบัติด้านความปลอดภัยคือใช้ โฟลเดอร์ทำงานเฉพาะ (ไม่ใช่ Documents/Downloads/Desktop ตรง ๆ) เพื่อจำกัดขอบเขตไฟล์ที่ Cowork เข้าถึง

# รวมไฟล์ที่เกี่ยวข้องไว้โฟลเดอร์เดียว เช่น
/work/q-report/
  ├── sales-jan.csv
  ├── sales-feb.csv
  └── sales-mar.csv

ใน Claude Desktop เลือกโหมด Cowork → "Work in a project" แล้วชี้ไปที่โฟลเดอร์นี้

2

มอบงานแล้วให้ Cowork ถาม clarify

สั่งงานเป็นเป้าหมายระดับงาน Cowork จะ ถามคำถามก่อนเริ่ม ถ้าโจทย์ยังกำกวม — ตอบให้ชัดแล้วมันจะวางแผนเอง

"รวมไฟล์ยอดขาย 3 เดือนในโฟลเดอร์นี้ ทำรายงานสรุปไตรมาส:
ยอดรวม, การเติบโตเทียบเดือนก่อน, สินค้าขายดี 5 อันดับ
และไฮไลต์ความผิดปกติ — ส่งเป็นเอกสารสรุปให้ฉันรีวิว"
3

ดู Cowork ลงมือ + steer กลางทางได้

Cowork จะอ่านไฟล์ คำนวณ และเริ่มร่างเอกสาร ระหว่างทางถ้าทิศไม่ตรง พิมพ์แทรกเพื่อปรับได้ทันที

"ขอกราฟแท่งเปรียบเทียบยอดรายเดือนด้วย และจัดหัวข้อตามภูมิภาค"
4

รับ deliverable (artifact) มารีวิว

ผลลัพธ์ออกมาเป็น ไฟล์ (artifact) ตรวจเหมือนรับงานจากเพื่อนร่วมงาน — ถ้าจะให้ส่งออก/แชร์ Cowork จะ ถามก่อนเสมอ

✅
ได้รายงานไตรมาสพร้อมกราฟและไฮไลต์ — จากไฟล์ดิบหลายไฟล์ จบในเซสชันเดียว
5

ทำให้เร็วขึ้นรอบหน้า (ออปชัน)

ถ้าทำรายงานแบบนี้ประจำ ให้บันทึกเป็น Skill (playbook) หรือใช้ global instructions เพื่อไม่ต้องอธิบายรูปแบบใหม่ทุกครั้ง และตั้ง /schedule ให้ทำตามรอบได้

📋 Prompt สรุป — คัดลอกไปใช้ได้เลย

🤝 Master Prompt — งานเอกสารหลายขั้นใน Cowork
ทำงานในโฟลเดอร์โปรเจกต์นี้ (มีไฟล์ข้อมูลดิบหลายไฟล์)

เป้าหมาย: รวมข้อมูลทั้งหมดเป็นรายงานสรุป 1 ไฟล์ ที่มี
- ตัวเลขรวม + การเปลี่ยนแปลงเทียบช่วงก่อน
- 5 อันดับเด่น + ไฮไลต์ความผิดปกติ
- กราฟประกอบ

ถ้ามีอะไรกำกวมให้ถามก่อน แล้ววางแผน-ลงมือ-ส่งเป็นเอกสารให้ฉันรีวิว
อย่าส่งออก/แชร์ให้ใครจนกว่าฉันจะอนุมัติ

🎯 สิ่งที่ได้จากเคสนี้

  • เลือกงานที่เหมาะกับ Cowork: หลายขั้นตอน + อิงไฟล์จริง
  • ใช้โฟลเดอร์ทำงานเฉพาะ และอาศัยที่ Cowork ถามก่อน send/share
  • มอบเป้าหมาย ปล่อยให้ clarify + วางแผน แล้ว steer กลางทางได้
  • ทำซ้ำเร็วขึ้นด้วย Skill / global instructions / /schedule
