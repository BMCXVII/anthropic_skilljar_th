AI Fluency: Framework & Foundations

คอร์สแกนของตระกูล AI Fluency — เรียนรู้กรอบ 4D Framework เพื่อทำงานร่วมกับ AI อย่าง effective, efficient, ethical และ safe

⏱️ ~20 บท · Foundational 🎯 หัวใจคือ 4D Framework 📌 สถานะ: ลงทะเบียนแล้ว

เรียบเรียงตามลำดับบทเรียนจริง: ทำไมต้อง AI Fluency → พื้นฐาน Generative AI → แล้วลงลึกทีละ D ทั้งสี่ (Delegation, Description, Discernment, Diligence)

🧭 ทำไมต้อง AI Fluency

AI Fluency คือทักษะการทำงานร่วมกับ AI ให้ได้ผลดีและมีความรับผิดชอบ มี 3 วิธีหลักในการใช้ AI: Automation (ให้ทำแทน), Augmentation (ทำร่วมกัน), และ Agency (ให้ทำเองอย่างมีอิสระระดับหนึ่ง) ศูนย์กลางของทั้งหมดคือ 4D Framework

▶ วิดีโอ: แนะนำ AI Fluency + 4D

⚙️ พื้นฐาน Generative AI

Generative AI สร้างเนื้อหาใหม่ได้เพราะ 3 สิ่งมาบรรจบกัน: transformer architecture + ข้อมูลฝึก (data) + พลังประมวลผล (compute) และผ่านสองช่วงคือ pre-training แล้ว fine-tuning

▶ พื้นฐาน Generative AI / LLM / transformer

▶ 3 สิ่งที่ทำให้ LLM เป็นไปได้ + pre/fine-tuning

① Delegation — เลือกว่างานไหนให้ใคร

ตัดสินใจว่างานไหน ทำเอง / ทำร่วมกับ AI / ให้ AI ทำ ประกอบด้วย Problem Awareness (เข้าใจปัญหา), Platform Awareness (รู้ว่าเครื่องมือทำอะไรได้) และ Task Delegation (มอบหมายงานให้เหมาะ)

▶ วิดีโอ: Delegation — Problem/Platform Awareness + Task Delegation

② Description — สื่อสารกับ AI ให้ชัด

คือศิลปะของการบอก AI ว่าต้องการอะไร แบ่งเป็น Product (ผลลัพธ์ที่อยากได้), Process (วิธีให้ทำ) และ Performance (มาตรฐาน/บทบาท) Description มาคู่กับเทคนิค prompting 6 อย่าง เช่น ให้ context, ยกตัวอย่าง, ระบุข้อจำกัด, แตกเป็นขั้นตอน

▶ Description: Product/Process/Performance

▶ 6 เทคนิค prompting

③ Discernment — ประเมินผลอย่างมีวิจารณญาณ

ประเมินทั้ง output (Product), วิธีคิด (Process) และ พฤติกรรม (Performance) ของ AI — ไม่เชื่อทุกอย่างทันที แต่ตรวจสอบเป็น

▶ วิดีโอ: Discernment — Product/Process/Performance

🔁
Description–Discernment loop สองตัวนี้ทำงานเป็นวงจร: สื่อสาร → ประเมิน → ปรับคำสั่ง → ประเมินใหม่ จนได้ผลที่ต้องการ

④ Diligence — รับผิดชอบต่อการใช้ AI

ครอบคลุมความรับผิดชอบ 3 ด้าน: Creation (สร้างอย่างมีความรับผิดชอบ), Transparency (โปร่งใสว่าใช้ AI), และ Deployment (นำไปใช้อย่างรอบคอบ)

▶ วิดีโอ: Diligence — Creation/Transparency/Deployment

🎯 สรุปใจความสำคัญ

  • AI Fluency = ทำงานกับ AI อย่าง effective, efficient, ethical, safe โดยมี 4D เป็นแกน
  • 3 วิธีใช้ AI: Automation, Augmentation, Agency
  • Delegation เลือกงานให้เหมาะ · Description สื่อสารชัด · Discernment ประเมินผล · Diligence รับผิดชอบ
  • Description กับ Discernment ทำงานเป็น loop เพื่อขัดเกลาผลลัพธ์
  • Generative AI = transformer + data + compute ผ่าน pre-training + fine-tuning
